# Presence editor art direction — version 3

## Purpose

The presence editor must preserve premium design quality rather than reducing every template to the same generic set of blocks. Version 3 expands the section vocabulary and separates global theme choices from section-level art direction.

## Screenshot regressions corrected

- Preview service cards no longer render the prototype phrase **Try booking**.
- Section headings and body copy inherit the section foreground colour instead of being overwritten by the global theme colour.
- Pill-shaped buttons no longer make contact panels and other large content surfaces fully oval.
- Mobile service cards preserve readable contrast and wrap booking actions safely.

## New section types

### Meet the owner

A dedicated practitioner/founder profile with:

- portrait, profile-card, and editorial layouts
- professional title and biography
- credentials and trust signals
- personal quote and signature treatment
- optional booking call to action

The general About section is now free to describe the practice, business story, philosophy, or environment.

### How it works

A configurable process section with:

- numbered steps
- vertical timeline
- modular cards
- two, three, or four-column desktop layouts
- a single-column mobile journey

## Expanded hero controls

The editor exposes the same art-direction vocabulary used by premium templates:

- cover, split, and editorial structure
- classic, luxury, editorial, bold, professional, creative, and education styles
- full-bleed, framed, arch, offset, and collage image treatments
- trust labels
- decorative background words
- optional floating story card
- optional scroll cue

## Gallery default

New gallery sections begin as a carousel. Grid, masonry, carousel, and editorial mosaic remain selectable in the editor.
